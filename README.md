# python_exam Natural Language Processing


kort description:
Et projekt der omhanlder Natural Language Processing. Vi vil komme ind på data cleaning, sortering af
stopwords, og tekstanalyse fra de 4 taler.
Vi kigger særligt på om dronningen og statministeren har forskellige emner de bringer op i deres
nytårstaler.


List of uses technologies
Pandas.
Sentida.
NLTK.
Matplotlib.
BeautifulSoup
WordCloud
ipynb

Installation guide:

pip install sentida
Kør NLTK-download.py


User guide:

1. kør d_tale.py and s_tale.py for at  fetche og download talerne.
2. kør wc_visual.ipynb for at se wordclouds og statistikker for talerne.
