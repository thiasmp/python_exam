Et projekt der omhanlder Natual Language Processing. Vi vil komme ind på data cleaning, sortering af
stopwords, og tekstanalyse fra de 4 taler.
Vi kigger særligt på om dronningen og statministeren har forskellige emner de bringer op i deres
nytårstaler.

1. Scrape de her hjemmesider og få dem gemt lokalt på computeren.
2. data cleaning og fjerne stopwords.
3. Analysere de forskellige taler, og lave word clouds ud af dem.
4. Sentiment Analysis Analyze sentiment values
5. Plot sentiment values
